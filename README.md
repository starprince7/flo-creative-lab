# FloCreativeLab

A professional video editing and post-production agency website built with Next.js.

## About FloCreativeLab

**Project Type:** Website  
**Location:** Lagos, Nigeria

The Founder of FloCreativeLab is a professional video editor specializing in:
- **Polished Music Videos** — Crafting visual experiences that amplify the power of music
- **TV Commercials** — Creating compelling brand stories that drive results
- **Visual Effects (VFX)** — Bringing creative visions to life with cinematic excellence

### About Description

> Driven by a passion for transforming raw footage into refined, cinematic visuals, I work closely with artists and brands to deliver edits that feel intentional, polished, and aligned with the story being told.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS, SCSS
- **Animations:** GSAP
- **Content:** Markdown-based services and projects

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` — Next.js App Router pages
- `/components` — Reusable UI components
- `/data` — Markdown content for services and projects
- `/public` — Static assets (images, fonts)

## Services

- **Music Videos** — `/flim-making/services/music-videos`
- **TV Commercials** — `/flim-making/services/commercials`
- **VFX** — Visual effects and motion graphics
